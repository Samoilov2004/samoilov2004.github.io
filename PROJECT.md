# Personal Website

## Goal
Build a modern personal website hosted on GitHub Pages.

## Audience
- Recruiters
- ML Engineers
- Startup founders
- Open source contributors

## Tech Stack
- HTML
- CSS
- JavaScript
- No framework unless necessary
- Deploy via GitHub Pages

## Style
- Minimal
- Dark mode
- Fast loading
- Responsive
- Apple-inspired design

## Pages
- Home
- About
- Projects
- Blog
- Contact

## Priorities
1. Performance
2. Accessibility
3. SEO
4. Mobile first
5. Easy maintenance

## Never
- Don't use heavy libraries.
- Don't use jQuery.
- Don't generate unnecessary code.
